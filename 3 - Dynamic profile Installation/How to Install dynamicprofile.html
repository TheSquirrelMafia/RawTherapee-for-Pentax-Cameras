What does the dynamicprofile.cfg file do?

It’s basically is a file that has all the rules on how to automatically apply the correct PP3 file to the RAW image automatically when you open it up in RawTherapee. This is completely optional.

How do I install the dynamicprofile.cfg file?

Go to:

C:\Users\<whatever your user name is>\AppData\Local\RawTherapee<whateverotherversioniscalledhere>

You'll need administrator settings to paste it in there. Overwrite the file. The folder will more than likely be hidden & you'll probably need to enable viewing it in the Windows Explorer settings. If that folder does not exist, you’ll need to first run RawTherapee & then close it. The folder will now be there.

For you Linux guys, these would go in the .config folder, but you’ll have to probably change some lines of code or fix the links within RawTherapee. Same with you Mac folks.

Once you have placed the dynamicprofile.cfg file where it belongs, open up RawTherapee & go to:

Preferences > Image processing

Change For raw Photos: to (Dynamic)

That's it.

From there on, whenever you open up a RAW file, RawTherapee will automatically use the correct PP3 setting to use on its own according to the ISO setting & other settings in the image.
