What are DCP files?

These files have the same color settings that your Pentax camera has. They are required for the PP3 files that use the Pentax Image tones. These work better on recent Pentax cameras. From the K-5 to the K-1II. They’re not really meant for the older cameras like the K10D, K-7, & related, but you can give them a try.

How do I install the DCP files?

  • Copy the Camera profiles directory to wherever you please.

  • Open up RawTherapee, go to Preferences > Image Processing & scroll down to Camera profiles directory.

  • Point it that to the Camera profiles directory you just copied into your system.

  • You do this on Windows, Linux, and/or Mac.

If you dual boot Windows & Linux, you can create a folder in Windows & then go to Linux & point Camera profiles directory in RawTherapee to the Camera profiles directory in Windows to save storage space. That or you can create a new directory in your Linux system & follow the steps above.

There are multiple PP3 files that use those .dcp files to recreate the Pentax image tones. If they don’t work for you, there is probably an error in the way you installed them.

You can add more .dcp files here as well. Just name them accordingly.
